# React 19 Practice Playground

This scaffold contains practice files for learning modern React (React 19).
It is intentionally *incomplete*: each file leaves TODOs so you can implement the React logic and practice.

## How to run
1. unzip the project
2. run `npm install`
3. run `npm run dev`
4. open http://localhost:5173
